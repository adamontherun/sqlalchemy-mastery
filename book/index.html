The database toolkit that runs half the Python world, explained properly: engines and connection pools, the Session and its unit of work, querying that maps cleanly onto the SQL you already know, the async layer and its sharp edges, and how it all fits into a real FastAPI service.

Every chapter pairs with runnable examples and a challenge with failing tests for you to make pass — all against a real PostgreSQL 17 running in Docker. Start with docker compose up -d && uv sync, then read on.

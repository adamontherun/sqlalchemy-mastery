Chapter 4 · Part II — The ORM

Declarative models

A mapped class is a Python class, a table definition, and a type-checked schema document all at once. Modern SQLAlchemy makes the declaration read like the annotations you already write everywhere else.

The modern shape

Three ideas carry the whole system:

  • Mapped[T] declares the Python type, and SQLAlchemy derives the SQL type: int → INTEGER, str → VARCHAR, datetime → TIMESTAMP, Decimal → NUMERIC, an enum.Enum → a native PostgreSQL ENUM. Mapped[str | None] means nullable, so your type checker and your schema now agree by construction.
  • mapped_column() adds what annotations can't say: lengths, uniqueness, indexes, defaults, constraints. When you have nothing to add, omit it entirely (see nickname).
  • Nothing new happens underneath: the class generates exactly the Table you built by hand in Chapter 3. It's right there at User.__table__, and Base.metadata is the same MetaData you already know.
⏭ 2.1 watch

If you've seen declarative_base() (a function, not a class) or models built from bare Column attributes, that's 1.x style. It still runs in 2.0, but it's untyped and slated for the history books. New code should always subclass DeclarativeBase; 2.1 continues to modernize around the annotation-driven mapping (dataclass integration gets particularly nice there).

Defaults: Python-side, server-side, and why it matters

default= is filled in by SQLAlchemy when your Python code inserts a row and didn't set the attribute. server_default= goes into the CREATE TABLE itself, as DEFAULT now(), so it applies to every insert, including ones from psql, cron jobs, and services that aren't yours. For data integrity facts like timestamps, prefer the server side. (There's also onupdate= for last-modified columns.)

⚠ Gotcha

default=datetime.utcnow(), with parentheses, is the classic self-inflicted wound: it evaluates once at import, and every row gets the timestamp of your last deploy. Pass the function (default=datetime.utcnow), or better, use server_default=func.now() and never think about it again.
